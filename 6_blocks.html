Blocos / Divs

Elementos block-level

Sempre iniciam uma nova linha e tomam toda a largura da página.

Exemplo de elemento block-level:
A tag < div > não requer nenhum atributo, mas geralmente são utilizados os atributos:
id,
class,
style.
Outras tags block-level: < hr > < table > < h1 > ... < h6 > < video > < p > etc...

Elementos inline-level

Não iniciam uma nova linha e tomam apenas o tamanho necessário/definido.

Exemplo de elementos inline-level. A tag < span > não requer nenhum atributo, mas geralmente são utilizados os atributos: id, class, style.
Outras tags inline-level: < a > < b > < sub > < br /> etc...
